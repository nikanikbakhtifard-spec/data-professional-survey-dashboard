
Data Professional Survey Analysis Dashboard

📊 Project Overview


This project involves an end-to-end data analysis process—from cleaning raw survey data in Excel to creating an interactive dashboard in Power BI. The goal was to visualize the professional landscape of data roles, focusing on salary trends, programming preferences, and job satisfaction.
🔍 Key Insights from the Dashboard:
• Demographics: Analyzed data from 630 survey takers with an average age of 29.87.
• Salary Trends: Data Scientists and Data Engineers command the highest average salaries compared to other roles.
• Tech Stack: Python remains the dominant programming language among data professionals, followed by R.
• Job Satisfaction: On average, professionals rated their Work/Life Balance (5.74/10) higher than their Salary Satisfaction (4.27/10).
• Global Reach: Significant participation from the US, India, UK, and Canada.
🛠️ Tools & Technologies
• Excel: Used for initial data exploration and understanding the raw dataset.
• Power BI (Power Query):
• Data Cleaning: Handled missing values, standardized country names, and formatted salary columns.
• Data Transformation: Created custom columns and cleaned the "Favorite Programming Language" field.
• DAX: Used to create measures for averages and counts.
📈 Dashboard Features
• Treemap: Visualizing the distribution of survey takers by country.
• Bar Charts: Comparing salaries across job titles.
• Donut Chart: Difficulty of breaking into the data field.
• Gauge Charts: Measuring happiness levels regarding salary and work-life balance.
• Stacked Bar Chart: Breakdown of programming language popularity by job title.
🚀 How to Use
1. Download the .pbix file from this repository.
2. Open it in Power BI Desktop.
3. Interact with the filters to see how salary and satisfaction vary by region or role.
